props

Garfish.props is used to get the public content passed by the main application, to use shared data, or to pass specified content to child applications. Can be used with mobx.

Master application shared data#

import Garfish from 'garfish';import { observable, computed, action } from 'mobx';
class Store {  @observable price = 0;  @observable amount = 1;
  @computed get total() {    return this.price * this.amount;  }
  @action.bound  increment() {    this.amount++;  }}
Garfish.run({  basename: basePath,  domGetter: '#subModule',  apps,  props: {    store: new Store(),  },});

Sub-application fetching data#

const store = window.Garfish.props.store;const price = store.price;const amount = store.amount;store.increment();

Or get it in the provider#

export function provider({ dom, basename, store }) {  return {    render() {},    destroy() {},  };}
